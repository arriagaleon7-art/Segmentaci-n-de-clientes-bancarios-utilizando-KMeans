# Segmentaci-n-de-clientes-bancarios-utilizando-KMeans
Segmentación de una base de 10,000+ clientes bancarios utilizando el algoritmo de Machine Learning No Supervisado, KMeans. A través de un  proceso de EDA, ingeniería de variables y análisis demográfico, se identificaron 5 perfiles de comportamiento clave para optimizar la toma de decisiones  y la gestión de riesgo crediticio.
